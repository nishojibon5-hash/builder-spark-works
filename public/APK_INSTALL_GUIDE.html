üîß Troubleshooting Common Issues

"App not installed" error:

  • Enable "Install from Unknown Sources" permission
  • Uninstall any previous version first
  • Restart device and try again
  • Check sufficient storage space

"For security, your phone is not allowed to install apps..." error:

  • Go to Settings > Privacy > Install Unknown Apps
  • Select your browser or file manager
  • Enable "Allow from this source"

App crashes or won't open:

  • Clear app data: Settings > Apps > LoanBondhu > Clear Data
  • Restart device
  • Ensure stable internet connection
  • Update Android WebView if available
